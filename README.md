# 🚀 AI Career Navigator
Identify your skill gaps and get a personalized AI roadmap to your dream job.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js & Express
- **AI Engine:** Google Gemini AI
- **Database:** MongoDB

## ⚙️ How to Run
1. Clone the repo
2. Install dependencies: `npm install` in both folders.
3. Add your `.env` with Gemini API Key.
4. Run `npm run dev` (client) and `node index.js` (server).
